# 简单文件管理器

`简单文件管理器`（`SimpleFileManager`）是一款面向 Windows 的开源文件管理软件。

它希望在保留 Windows 原生使用习惯的基础上，提供更高效、更清晰的文件浏览与整理体验。无论是日常查看文件、批量整理目录，还是在多个位置之间快速切换，它都强调直接、稳定和高效率。

## 这是什么

- 一款面向普通 Windows 使用者的桌面文件管理工具
- 一款支持多标签页和多面板工作的文件管理软件
- 一款尽量贴近 Windows 交互习惯的开源应用

## 主要功能

- 多标签页浏览
  可以像浏览器一样同时打开多个文件位置，减少频繁来回切换路径的成本。

- 多面板工作区
  支持单面板、双面板以及更多面板布局，方便在不同目录之间对照、整理和移动文件。

- 目录树导航
  左侧目录树可以快速展开和切换文件夹，适合在层级较深的目录中快速定位目标位置。

- 多种查看方式
  支持超大图标、大图标、中等图标、小图标、列表、详细信息列表、平铺、内容等多种查看模式，适应不同使用习惯。

- Windows 风格图标
  文件、文件夹、磁盘等图标尽量采用贴近 Windows 系统风格的显示效果，整体观感更自然。

- 本地文件基础操作
  支持常见的文件浏览、复制、移动、重命名、删除、新建文件夹等日常管理操作。

- 拖放整理
  可以直接把文件拖动到目标文件夹完成移动；按住 `Ctrl` 再拖动时，可执行复制操作。

- 详细信息查看与排序
  在详细信息视图下，可以按名称、类型、大小、修改时间等维度查看和整理文件。

## 适合哪些场景

- 管理下载目录、文档目录、工作目录
- 在两个或多个文件夹之间频繁复制或移动文件
- 同时打开多个目录进行对照查看
- 希望获得比传统单窗口文件浏览更高效的使用方式
